# cookiecutter-poetry
Cookiecutter template configured with the following:

- poetry
- pytest
- black
- bandit
- pyinstaller
- click

A Makefile has been included so you don't have to remember commands.

## Usage:

    cookiecutter https://github.com/timhughes/cookiecutter-poetry.git
